# Calculator

## Description
A simple command-line calculator that performs basic arithmetic operations. Users can input two numbers and select an operator to get the result.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Input validation for numeric values
- Error handling for invalid operators

## Usage
1. Run the script: python calculator.py
2. Select an operator: + - * /
3. Enter the first number
4. Enter the second number
5. The result will be displayed

## Requirements
- Python 3.x
