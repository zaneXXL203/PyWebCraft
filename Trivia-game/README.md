# Python Trivia Game

## Description
An interactive Python trivia game that tests users' knowledge of Python programming concepts. The game asks random questions and keeps score.

## Features
- 8 different Python trivia questions
- Randomly selects 5 questions per game
- Displays correct answers when user answers wrong
- Keeps track of score
- Shows final score and percentage
- Case-insensitive answer matching
- Simple command-line interface

## Questions Cover
- Function definitions
- Data types (float, list, tuple)
- Comments
- Built-in functions (len())
- Exception handling
- Type checking
- Immutable data types
- Module imports

## Usage
1. Run the script: python "Trivia game.py"
2. Answer each question when prompted
3. Enter your answer in lowercase or uppercase
4. Receive immediate feedback on correctness
5. See the correct answer if you're wrong
6. Final score displays at the end

## Scoring
- Correct answer: +1 point
- Wrong answer: 0 points
- Final score shown as: SCORE: X/5

## Requirements
- Python 3.x
