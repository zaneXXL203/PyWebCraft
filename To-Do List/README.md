# To-Do List Manager

## Description
A command-line todo list application that allows users to create, view, and mark tasks as complete. Tasks are stored in a JSON file for persistence.

## Features
- View all tasks with their completion status
- Add new tasks
- Mark tasks as complete
- Persistent storage using JSON
- Clear task descriptions
- Status indicators ([pending...] or [completed])
- Input validation
- Error handling

## Menu Options
1. View Tasks - Display all tasks with status
2. Add Task - Create a new task
3. Complete Task - Mark a task as finished
4. Exit - Close the application

## Usage
1. Run the script: python "to-do list.py"
2. Choose an option from the menu
3. Follow the prompts to manage your tasks
4. Tasks are automatically saved to to-do list.json

## Task Status
- [pending...] - Task not yet completed
- [completed] - Task has been finished

## Storage
Tasks are stored in JSON format for easy backup and portability.

## Requirements
- Python 3.x
