# Dice Game

## Description
A simple interactive dice roller game where users can roll a dice (1-10) as many times as they want and see the results.

## Features
- Roll a 10-sided dice
- Display random number (1-10) after each roll
- Continuous gameplay with option to quit
- Simple yes/no interface
- Input validation

## Usage
1. Run the script: python dice.py
2. Answer the prompt "roll a dice (yes/no):"
3. If you choose "yes", a random number between 1-10 will be rolled
4. Choose "no" to exit the game

## Requirements
- Python 3.x
