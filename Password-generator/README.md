# Password Generator

## Description
A secure password generator that creates customizable passwords with optional uppercase letters, special characters, and digits. Users can specify password length and character types.

## Features
- Customizable password length (minimum 4 characters)
- Optional uppercase letters
- Optional special characters
- Optional digits
- Randomized character arrangement
- Always includes lowercase letters
- Validates minimum password length

## Usage
1. Run the script: python passwordgenerator.py
2. Enter desired password length
3. Choose character types:
   - Include uppercase letters? (yes/no)
   - Include special characters? (yes/no)
   - Include digits? (yes/no)
4. Your generated password will be displayed

## Example
Enter PASSWORD LENGTH: 12
UPPERCASE(yes/no): yes
SpecialCharacters(yes/no): yes
Digit(yes/no): yes

## Requirements
- Python 3.x
