# Add a Message

## Description
A simple text file manager that allows users to create text files and add messages to them. Users can specify a custom filename or use the default "default.txt" file.

## Features
- Create new text files with custom names
- Add messages to text files
- Automatic default file creation if no filename is specified
- Simple command-line interface

## Usage
1. Run the script: python message.py
2. Enter the filename (or press Enter to use "default.txt")
3. Enter your message
4. The message will be saved to the specified file

## Requirements
- Python 3.x
