# Alarm Clock

## Description
A command-line alarm clock application that allows users to set an alarm at a specific time. When the alarm time is reached, it plays an alarm sound and displays a wake-up message.

## Features
- Set alarm for a specific time (HH:MM:SS format)
- Real-time clock display showing current time
- Audio alert using pygame mixer
- Wake-up notification with emoji

## Usage
1. Run the script: python AlarmClock.py
2. Enter the alarm time in HH:MM:SS format (e.g., 14:30:00)
3. The program will monitor the time and trigger the alarm when the set time is reached
4. The alarm sound will play when triggered

## Requirements
- Python 3.x
- pygame library (pip install pygame)
- alarm.mp3 file in the same directory
