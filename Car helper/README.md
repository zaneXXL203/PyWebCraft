# Car Helper

## Description
An interactive car control simulator that demonstrates simple state management. Users can control a virtual car with start/stop commands and get help information.

## Features
- START: Start the car engine
- STOP: Stop the car engine
- HELP: Display available commands
- QUIT: Exit the program
- State validation (prevents starting an already started car, etc.)
- User-friendly command prompts

## Usage
1. Run the script: python "simple car helper.py"
2. Enter commands at the prompt:
   - Type help to see all available commands
   - Type start to start the car
   - Type stop to stop the car
   - Type quit to exit the program

## Requirements
- Python 3.x
