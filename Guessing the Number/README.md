# Guessing the Number

## Description
A number guessing game where the computer picks a random number between 1-100, and the player tries to guess it with feedback on whether their guess is too high or too low.

## Features
- Computer generates random number (1-100)
- Player attempts to guess the number
- Feedback on each guess (too high/too low/correct)
- Unlimited guesses until correct answer
- Displays the correct number upon winning
- Input validation

## Gameplay
1. Computer picks a random number between 1-100
2. Player enters their guess
3. Game provides feedback:
   - "TOO HIGH!" if guess is greater than the number
   - "TOO LOW!" if guess is less than the number
   - Congratulations message if guess is correct
4. Game ends when player guesses correctly

## Usage
1. Run the script: python "guess the number.py"
2. Follow the prompts to enter your guesses
3. The game will continue until you find the correct number

## Requirements
- Python 3.x
