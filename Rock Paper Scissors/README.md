# Rock Paper Scissors

## Description
A classic Rock Paper Scissors game where players can play against the computer. The game uses emojis to make it visually appealing and includes win/lose/tie detection.

## Features
- Play against the computer AI
- Visual representation using emojis:
  - R (Rock): 
  - P (Paper): 
  - S (Scissors): 
- Automatic win/loss/tie detection
- Quit option to exit the game
- Input validation
- Score feedback

## Gameplay Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Tie

## Usage
1. Run the script: python "rock paper and scissors.py"
2. Enter your choice at the prompt:
   - R for Rock
   - P for Paper
   - S for Scissors
   - Q to Quit
3. The computer makes its choice
4. Results are displayed with emojis
5. Continue playing or type Q to exit

## Requirements
- Python 3.x
