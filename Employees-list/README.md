# Employees List

## Description
A program that manages employee records and exports them to different file formats. Currently supports CSV export with sample employee data from the Krusty Krab (SpongeBob SquarePants reference).

## Features
- Store employee data with name, age, and job title
- Export employee data to CSV format
- Customizable file paths for output
- Sample data pre-loaded (SpongeBob, Patrick, Sandy)
- Error handling for existing files

## Data Structure
Employee records include:
- Name
- Age
- Job Title

## Usage
1. Run the script: python Employees.py
2. Enter your message (optional)
3. Specify the file path and format (e.g., "employees.csv")
4. The file will be created with employee data

## Files Included
- employees.json: Sample employee data
- Krusty krab.txt: Reference data
- What they do.csv: Job descriptions

## Requirements
- Python 3.x
